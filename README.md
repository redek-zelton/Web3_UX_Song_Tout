# Web3Js
## Start
### pour le client
Pour commencer le projet :
```sh
npm install
npm install axios
npm install web3@latest ( il faut avoir la version 1.3.4)
npm run serve
```
### pour le serveur
Dans une seconde console :
```sh
cd server
npm i
npm run dev
```
